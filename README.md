# Digital-Systems-Project

In order to run this software, the follwoing software packages need be installed:

Python 3.6 – programming language used
	https://www.python.org/downloads/release/python-360/

PyQt5 – the graphical user interface library
	https://pypi.org/project/PyQt5/

opencv-python – computer vision library
	https://pypi.org/project/opencv-python/

keras – deep learning library
	https://pypi.org/project/Keras/

numpy – scientific computing library
	https://pypi.org/project/numpy/

imutils – edge detection library
	https://pypi.org/project/imutils/

face_recognition – face recognition library build using dlib and deep learning
	https://pypi.org/project/face-recognition/

threading – threading package 
	prebuilt into Python 3.6

random – number manipulation library
	prebuilt into Python 3.6

collections – data types library
	https://pypi.org/project/collections2/
